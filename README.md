<h1>🕵️‍♂️ Purrlock Holmes: Cat Meme Picker</h1>
<p>A simple web application that "investigates" and retrieves the perfect cat meme to match your emotions.</p>
<h3>✨ Features</h3>
<ul>
  <li><b>Emotion Selection:</b> Choose from a list of radio buttons to find a cat that matches your mood (e.g., Hungry, Sad, Confused).</li>
  <li><b>GIF Filter:</b> A checkbox option to specifically search for animated memes.</li>
  <li><b>Modal Display:</b> The selected meme opens in a focused modal window for a clean viewing experience.</li>
  <li><b>Responsive Design:</b> Works smoothly on different screen sizes.</li>
</ul>
<h3>🚀 How It Works</h3>
<ol>
  <li><b>Select a Mood:</b> Click one of the emotion radio buttons.</li>
  <li><b>Toggle GIFs (Optional):</b> Check the "Animated GIFs only" box if you want a moving image.</li>
  <li><b>Find the Cat:</b> Click the "Get Image" button to trigger the modal.</li>
